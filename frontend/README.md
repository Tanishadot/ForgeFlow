# Factory Copilot Dashboard (Frontend)

This folder contains a Vite + React + TypeScript dashboard with TailwindCSS.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

The dev server runs on `http://localhost:3000` and proxies `/explain` and `/api` calls to `http://localhost:8000` by default.

Pages:
- Dashboard
- Copilot
- Simulation

Widgets:
- Orders Table
- Machine Status
- Inventory Status
- Production Schedule
- Gantt Chart

Theme: NVIDIA-style dark theme using `nvidia` color palette in Tailwind config.
